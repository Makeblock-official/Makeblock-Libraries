\begin{DoxyParagraph}{Description}
Stop forcing power onto the bus. You only need to do this if you used the \textquotesingle{}power\textquotesingle{} flag to \doxylink{class_me_one_wire_a50f690ef7f97a23844da2b89e78232d7}{write()} or used a \doxylink{class_me_one_wire_a71ff538d319b4492e3002e027b07e9fa}{write\+\_\+bit()} call and aren\textquotesingle{}t about to do another read or write. You would rather not leave this powered if you don\textquotesingle{}t have to, just in case someone shorts your bus.

\begin{DoxyParagraph}{Function}
search 
\end{DoxyParagraph}
\begin{DoxyParagraph}{Description}
Perform a search. If this function returns a \textquotesingle{}1\textquotesingle{} then it has enumerated the next device and you may retrieve the ROM from the Me\+One\+Wire\+::address variable. If there are no devices, no further devices, or something horrible happens in the middle of the enumeration then a \textquotesingle{}0\textquotesingle{} is returned. If a new device is found then its address is copied to new\+Addr. Use \doxylink{class_me_one_wire_a83f6c2c88d9f2fa5c4aabb7f540bc6ac}{Me\+One\+Wire\+::reset\+\_\+search()} to start over. you can get the algorithm from doc\textbackslash{}\+Me\+\_\+\+Temperature\textbackslash{}\+Datasheet\textbackslash{}1-\/\+Wire\+\_\+\+Search\+\_\+\+Algorithm.\+pdf 
\end{DoxyParagraph}
